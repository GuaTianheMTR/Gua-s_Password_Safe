# Gua-s_Password_Safe
这是@呱瓜多米 制作的使用浏览器（建议使用Microsoft Edge）本身的安全存储为安全机制的密码存储器。只要输入主密码，你就可以查看你在这个HTML内存储的密码。|This is a password storage tool created by @呱瓜多米, which uses the browser's built-in secure storage (Microsoft Edge is recommended) as its security mechanism. By entering the master password, you can view the passwords stored within this HTML file.
